RIMMF

RIMMF stands for "RDA in Many Metadata Formats". It is a tool for creating and editing data compliant with RDA. The latest version is RIMMF3.

  • Download RIMMF: Version 3
  • RIMMF3 User Guide
  • RIMMF Tutorials

RIMMF is used to create and augment r-balls, and is the main data entry tool for a Jane-athon.
