RDA element sets

Item properties

The Item properties element set consists of properties representing the attributes and relationships of the RDA Item entity. Each property in the element set has a domain of the class representing this entity.

Number of elements: 54
Namespace: http://rdaregistry.info/Elements/i/
Suggested prefix: rdai
Example curie (canonical)*: rdai:P40001
Example curie (lexical)*: rdai:identifierForTheItem
Changelog feed: Atom | RDF

*All RDA URIs have both an immutable canonical form and a 'readable', lexical form, which is subject to change (changes will be redirected).
