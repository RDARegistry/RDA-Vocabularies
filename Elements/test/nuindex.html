Turtle (text/turtle)

The map consists of shortcut mappings entailed from adding the Map from unconstrained ISBD properties to unconstrained properties derived from RDA to the ISBD elements element set, which includes mappings to the ISBD elements (unconstrained) element set.

For example:

# from ISBD elements:
isbd:P1001 reg:hasUnconstrained isbdu:P1001 .
# from http://metadataregistry.org/uri/profile/RegAp/ (reg):
reg:hasUnconstrained rdfs:subPropertyOf rdfs:subPropertyOf .
# entails:
isbd:P1001 rdfs:subPropertyOf isbdu:P1001 .
# from map of unconstrained ISBD to unconstrained properties derived from RDA:
isbdu:P1001 rdfs:subPropertyOf rdau:P60049 .
# entails:
isbd:P1001 rdfs:subPropertyOf rdau:P60049 .

The map is given in a terse triple language (ttl) serialization.
