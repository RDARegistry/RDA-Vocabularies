This section contains alignments between RDA element sets and other vocabularies. Alignments are given in the form of tables, usually in a comma separate variable file, containing general relationships that may ignore the precise semantics embedded in the vocabularies.

Alignments are the bases of maps. A map is a set of RDF triples representing the semantic relationship between two element sets or value vocabularies.
