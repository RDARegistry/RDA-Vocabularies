RDA element sets

RDA Entity properties

The RDA Entity properties element set consists of properties representing elements of the RDA Entity entity.

Each property in the element set:

  • has a domain of the class representing the RDA Entity entity.
  • is linked from its child datatype property in RDA Entity datatype properties by rdfs:subPropertyOf.
  • is linked from its child object property in RDA Entity object properties by rdfs:subPropertyOf.
Number of elements: 29
Namespace: http://rdaregistry.info/Elements/x/
Suggested prefix: rdax
Example curie (canonical)*: rdax:P00001
Example curie (lexical)*: rdax:relatedEntity

*All RDA URIs have both an immutable canonical form and a "readable", lexical form, which is subject to change (changes will be redirected).
