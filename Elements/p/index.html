RDA element sets

Place properties

The Place properties element set consists of properties representing attributes and relationships of the RDA Place entity.

Each property in the element set:

  • has a domain of the class representing the Place entity.
  • is linked from its child datatype property in RDA Place datatype properties by rdfs:subPropertyOf.
  • is linked from its child object property in RDA Place object properties by rdfs:subPropertyOf.
Number of elements: 17
Namespace: http://rdaregistry.info/Elements/p/
Suggested prefix: rdap
Example curie (canonical)*: rdap:P70001
Example curie (lexical)*: rdap:nameOfAPlace
Changelog feed: Atom | RDF

*All RDA URIs have both an immutable canonical form and a "readable", lexical form, which is subject to change (changes will be redirected).
