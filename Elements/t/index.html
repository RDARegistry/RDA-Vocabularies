RDA element sets

Timespan properties

The Timespan properties element set consists of properties that represent the attribute and relationship elements of the RDA Timespan entity.

Each property in the element set:

  • has a domain of the class that represents the Timespan entity.
  • is linked from its child datatype property in RDA Timespan datatype properties by rdfs:subPropertyOf.
  • is linked from its child object property in RDA Timespan object properties by rdfs:subPropertyOf.
Number of active elements: 57
Namespace: http://rdaregistry.info/Elements/t/
Suggested prefix: rdat
Example curie (canonical)*: rdat:P70001
Example curie (lexical)*: rdat:relatedWorkTimespan

*All RDA URIs have both an immutable canonical form and a "readable", lexical form, which is subject to change (changes will be redirected).
