This section contains maps to and from RDA element sets and value vocabularies. We use the term "map" to refer to a set of mappings. We use the term "mapping" to refer to a single relationship between two classes, properties, or concepts taken from different element sets and value vocabularies.

A map is a set of RDF triples representing the semantic relationship between two element sets or value vocabularies.

Some maps are based on alignments. An alignment is usually a comma separate variable file containing general relationships that may ignore the precise semantics embedded in the vocabularies.
