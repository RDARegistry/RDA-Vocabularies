This map relates codes from the MARC code list for relators to RDA relationship designators and relationship elements. The code list is available in Code Sequence.

The map is given in a terse triple language (ttl) serialization.

The subject of each mapping is the URI of a MARC relator code with no domain or range and typed as skos:Concept, rdfs:Property, owl:ObjectProperty, and mads:Topic.

The predicate of each mapping is a SKOS property with no domain or range.

skos:broadMatch is used rather than skos:exactMatch because there is no formal alignment between RDA and the MARC code list. This looser mapping can be augmented by skos:exactMatch at any point in the future without deprecation because skos:exactMatch is a sub-property of skos:closeMatch. skos:broadMatch is used for mrc:cng ("Cinematographer") and mrc:vdg ("Videographer") because they are narrower in scope that rdau:P60452 (designator "director of photography").

The object of each mapping is an RDA unconstrained property with no domain or range and typed as rdfs:Property. The property represents a relationhip element or relationship designator from Appendix I of the RDA Toolkit, not differentiated with respect to RDA entities.
