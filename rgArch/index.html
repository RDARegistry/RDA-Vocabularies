This section contains tools for using older versions of RDA Vocabularies.

Change of original namespace

The RDA Registry namespace was changed from an original namespace used for initial development of the RDA elements sets and value vocabularies.

The following maps relate elements and concepts from the original namespace, now deprecated, to the current namespace. Note that elements and concepts may have subsequently been deprecated in turn.

  • Map from deprecated to published RDA classes
  • Map from deprecated to published RDA properties
  • Map from bad to correct URIs for RDA Colour Content value vocabulary
  • Map from deprecated to published RDA Carrier Extent Unit concepts (Jul 2016)
  • Map from deprecated to published RDA Generation concepts (Jul 2016)
  • Map from deprecated to published RDA Layout concepts (Jul 2016)
  • Map from deprecated to published RDA Material concepts (Jul 2016)
  • Map from deprecated to published RDA Production concepts (Jul 2016)
  • Map from miscelleneous deprecated to published RDA value vocabulary concepts (Jul 2016)
