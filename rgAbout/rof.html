About the RDA Registry and vocabularies

RDA/ONIX Framework

The RDA/ONIX Framework for Resource Categorization provides sets of defined attributes of resource content and resource carriers and specified primary values for a subset of the defined attributes.

The Framework is used to derive content and carrier categories specified in RDA.

The Framework attributes and values are represented in the RDA Registry as:

  • RDA/ONIX Framework element set
  • RDA/ONIX value vocabularies
    • ROF Character
    • ROF Extension Mode
    • ROF Extension Requirement
    • ROF Extension Termination
    • ROF Housing Format
    • ROF Image Dimensionality
    • ROF Image Movement
    • ROF Interaction
    • ROF Intermediation Tool
    • ROF Revision Mode
    • ROF Revision Requirement
    • ROF Revision Termination
    • ROF Sensory Mode
    • ROF Storage Medium Format

The Framework is the basis of the RDA value vocabularies:

  • RDA Carrier Type
  • RDA Content Type
  • RDA Media Type

More specific RDA types use RDA refinements to RDA/ONIX value vocabularies:

  • ROF Character for RDA
  • ROF Form/Genre for RDA
  • ROF Intermediation Tool for RDA
  • ROF Storage Medium Format for RDA

The Framework is intended for use in a hub-and-spoke mapping architecture. Categories from each schema are mapped to concepts in the ROF value vocabularies, instead of directly to each other. Categories from two different schema can be mapped by matching the values of their ROF concepts.
